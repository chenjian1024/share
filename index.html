masonry 在ios10中布局的坑

解决更新ios10以后 使用masonry布局动态Cell出现约束重复和警告cell高度为0的问题

View on GitHub Download .zip Download .tar.gz

在 xcode7 中的代码是这么写

[self.lastChildView mas_makeConstraints:^(MASConstraintMaker *make) {
    make.left.right.mas_equalTo(self.contentView);
    make.top.equalTo(self.fromCircleBtn.mas_bottom).offset(RealPoint(20));
    make.height.mas_equalTo(RealPoint(20));
}];

[self.contentView mas_makeConstraints:^(MASConstraintMaker *make) {
    make.left.right.top.equalTo(self);
    make.bottom.equalTo(self.bgView.mas_bottom).offset(RealPoint(20));
}];

在xcode8中会出现一些约束报错的问题

xcode8中这么写可以解决就解决了

[self.bgView mas_makeConstraints:^(MASConstraintMaker *make) { make.left.right.mas_equalTo(self.contentView); make.top.equalTo(self.fromCircleBtn.mas_bottom).offset(RealPoint(20)); make.height.mas_equalTo(RealPoint(20)); make.bottom.equalTo(self.contentView).offset(RealPoint(-20));

}];

[self.contentView mas_makeConstraints:^(MASConstraintMaker *make) {
    make.left.right.top.bottom.equalTo(self);
}];
